"Jade Hotels Booking Management System"


TO ADD JADE HOTELS IN LOCALHOST

1. First download XAMPP

2. Extract the file and copy the 'hotel' folder

3. Paste inside the root directory/ where you install XAMPP local disk C: drive D: drive E: paste: (for xampp/htdocs) the 'hotel' file

4. Open PHPMyAdmin (http://localhost/phpmyadmin)

5. Create a database with name hotel_db

6. Import hotel_db.sql (file in jadehotels/database)

7. Run the script http://localhost/jadehotels


TO ACCESS ADMIN
